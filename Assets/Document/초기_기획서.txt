# 개요

### 게임 제목 (미정)

- **룬 키우기**
- 머지 요소를 룬이 아닌 ‘정령석’으로 하여 씹덕 요소를 섞는 방향으로도 생각 중.

### 장르

- 캐주얼 게임
- 방치형 머지 게임
- RPG

### 플랫폼

- 모바일 게임(Android, IOS)

### 타겟 유저

- 모바일 게임에 긴 시간을 투자하기 힘든 직장인.
- RPG 성장 요소를 좋아하는 플레이어.

### 로그라인

- 플레이어는 룬을 조합하고 새롭고 강한 스킬을 획득하여 점점 높은 스테이지로 올라가는 게임입니다.
- 다시 작성 필요

### 핵심 컨셉

- 머지 라는 컨셉 자체는 흔하지만, 이것을 플레이어의 스킬로써 전투에 직접적으로 활용되는 경우는 없었음.
- 머지를 통해 점점 새로운 룬을 발견하고 이것들을 전투에 직접 적용하고, 스킬들을 조합하여 플레이어만의 스킬 조합을 짜는 등의 다채로운 재미를 느낄 수 있음.

### 게임 핵심 매커니즘

- **머지(Merge) 시스템 기반 게임**
    - 같은 요소(오브젝트)를 **두 개 합치면 상위 단계로 진화**.
        - 예) 표창 키우기 - 같은 표창 2개를 합치면 다음 단계 표창 1개로 업그레이드 됨.
- **스테이지 기반 자동 전투 시스템**
    - 상단의 화면에서 플레이어가 자동으로 전투를 이어가며 클리어 시 다음 스테이지로 진입 함.
    - 스테이지 클리어 시, 일정량의 재화를 획득 가능함. (오프라인 시, 보상 누적 = 방치형 시스템)
- **머지 요소 = 스킬 슬롯 장착 아이템**
    - 머지 요소(룬)은 단순히 수집하는 게 아니라, 플레이어가 전투에 직접 사용.
        - 스킬 슬롯에 제작한 머지 요소(룬)을 끌어다 장착하여 플레이어의 스킬로써 사용할 수 있음.

# 게임 상세 설명

### 시놉시스 및 세계관

- 미정

### 핵심 게임 플레이

- 1단계 룬 생성. ⇒ 동일 룬을 조합하여 다음 단계의 룬 획득.
- 룬을 스킬 슬롯에 장착하여 활용 ⇒ 플레이어의 전투력이 증가하여 스테이지를 점점 원활하게 클리어
- 스테이지를 클리어하여 획득한 보상으로 각종 생성/전투 관련 요소들을 업그레이드.

### 주요 시스템 및 콘텐츠

- 상단 자동 전투 레이아웃
    - 왼쪽에 플레이어가 위치한 채로 오른쪽으로 이동하며 다가오는 적들을 격파.
    - 기본적인 전투 자체는 자동으로 진행됨. (캐릭터가 평타 사용)
    - 스테이지 존재. 해당 스테이지의 모든 적을 격파 시, 재화를 획득하고 다음 스테이지로 진입.
    - 10 스테이지마다 보스 등장.
    - 스테이지 클리어 실패 시, 바로 이전 스테이지로 진입.
        - 보스 스테이지에서 실패 시, 바로 이전 스테이지로 진입하며, 해당 스테이지를 반복.
        화면에 추가적인 ‘도전’ 버튼이 추가되어, 플레이어가 해당 버튼을 클릭 시 보스 스테이지로 이동 후 도전 시작.
- 중단 스킬 슬롯 레이아웃.
    - 플레이어가 스킬을 등록할 수 있는 공간.
    - ‘룬’을 터치해 이동시켜 슬롯에 가져다 놓을 시, 스킬이 등록됨.
    - 플레이어가 스킬이 등록된 슬롯을 클릭 시, 해당 스킬이 발동.
    - Auto로 설정해 놓을 시, 0.25초의 간격을 두고 왼쪽부터 우선순위로 하여 스킬이 발동. 
    + 쿨타임이 돌고 있지 않은 스킬이 우선으로 발동.
        - 스킬 예시) 슬롯 1. 평타(1초)  / 슬롯 2. 강타(5초) / 슬롯 3. 중타(2초) 일 시,
        ⇒ 평타 → 강타 → 중타 순서대로 스킬이 발동. 
        ⇒ (2초 뒤 쿨타임이 돈 후) 평타 → 중타 (강타는 쿨타임 중)
        - 스킬 예시) 슬롯 1. 평타(0.1초)  / 슬롯 2. 강타(5초)
        ⇒ 평타 만 나가고 강타는 발동되지 않음 
        (0.25초 딜레이 후, 스킬 발동 시점 기준 우선순위 조건에 따라 스킬이 발동)
        - 쿨타임을 잘 계산하여 슬롯 장착의 순서를 체계적으로 구성 시, 기대치 이상의 전투 효율을 뽑아낼 수 있음.
    - 최초 스킬 슬롯은 3개
        - 이후 업그레이드를 통해 슬롯 해제 가능
    - 스킬이 담긴 슬롯을 ‘꾹’하고 길게 누르면 해제 가능. 그대로 슬롯 위치 옴기기도 가능.
    - 프리셋 기능 존재 (다른 콘텐츠를 위한)
- 하단 메인 머지 콘텐츠 레이아웃
    - 룬이 생성되고 합치는 메인 과정을 진행하는 공간
    - 레이아웃의 오른쪽 하단에 위치한 버튼을 눌러 ‘1단계’ 룬을 생성 가능.
        - 생성 가능한 개수에 대해서는 충전 형식으로 구현. (10초당 1개 충전)
            - 예) 생성 가능 개수 10/10 ⇒ 3개 생성 ⇒ 7/10 ⇒ 20초 후 ⇒ 9/10
    - 보유 가능한 룬의 최대 개수 제한 (최초 10개)
- 메뉴 : 업그레이드
    - 전투 업그레이드
        - 공격력
        - 방어력
        - 체력
        - …
    - 특수 업그레이드
        - 최대 보유량 증가
        - 생성 쿨타임 감소
        - 최초 생성 시 랜덤으로 2단계
        - …
- 메뉴 : 주간 레이드 (하루 3회)
    - 월화 / 수목 / 금토 별로 3가지 레이드 존재.
    - 일 모든 레이드 플레이 가능.
    - ㅈㄴ쌤. 죽이지는 못함
        - 입힌 피해량에 비례하여 보상 지급
        - 현재 전투력이 어느정도 인지 테스트를 위한 콘텐츠
    - 스킬 구성) 높은 DPS의 스킬 구성 및 조합 유도.
- 메뉴 : 던전
    - 수많은 물량의 적이 몰려오는 특수 던전
    - 스킬 구성) 범위 및 유지력 스킬 구성 및 조합 유도
